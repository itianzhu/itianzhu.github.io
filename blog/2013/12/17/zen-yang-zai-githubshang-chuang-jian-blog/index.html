怎样在github上创建Blog

Dec 17th, 2013

必须的东西

1.安装git

2.建立github账号

3.在github上建立username.github.io为名的仓库

然后运行下面的shell命令

git clone git://github.com/imathis/octopress.git octopress
cd octopress
bundle install
rake install //安装默认模板
rake setup_github_pages //按照提示填入信息

打开source/_config.yml填入Blog信息

rake generate
rake deploy //按照提示填入信息
rake new_post["title"] //创建新的文章
rake generate
rake preview
rake deploy
